Submit a domain

Submit a domain or subdomain you control, then point its mail DNS here before sending it for review.

Set DNS like this:
A: Name: mail -> IPv4 address: loading...
MX: Name: @ -> Mail server: yourdomain.com -> Priority: 10

  • For @yourdomain.com: set MX Name = @, Mail server = mail.yourdomain.com.
  • For @mail.yourdomain.com: set MX Name = mail, Mail server = mail.yourdomain.com.
  • If you use Cloudflare, keep mail.yourdomain.com as DNS only.
